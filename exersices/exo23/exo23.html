Exercice 23

Créer une connexion avec la base de donnée.

Créer une table "book" avec les champs "id", "name"(string), "releaseDate"(date), "author"(string).

Créer également une classe "Book" avec les mêmes attributs.

Créer une fonction "createUser" qui prendra en paramètre un User et qui ira ajouter en base l'utilisateur avec les informations contenu dans l'objet.

Créer une fonction "readUser" qui prendra en paramètre un id et qui retournera un objet User avec les informations correspondantes.

Créer une fonction "getAllUsers" qui retournera une liste d'objets User qui correspondant à chaque entrée en base.

Créer une fonction "updateUser" qui prendra en paramètre un User (si l'utilisateur n'a pas d'id renvoyer une erreur. Voir "throw new Exception"). Vous devrez mettre à jour les différents champs en base avec ceux passer par l'objet User en paramètre.

Créer une fonction "deleteUser" qui prendra en paramètre un id et qui supprimera l'entrée correspondante en base.
