Exercice 22

Créer un formulaire avec les champs lastname, firstname, email, age.

Créer une classe User avec les mêmes attributs.

À la validation du formulaire vous devrez créer un nouvel objet User en le remplissant avec les informations du formulaire.

Vous devrez enfin afficher votre objet sur votre page HTML avec toutes les informations qu'il contient.
