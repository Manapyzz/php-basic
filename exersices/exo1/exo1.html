Exercice 1

Créer une page PHP qui contiendra les variables suivantes:

  • variable avec pour nom “firstname” et qui aura comme valeur votre prénom (string).
  • variable avec pour nom “age” et qui aura comme valeur votre age (int).
  • variable avec pour nom “height” et qui aura comme valeur votre taille (float).
  • variable avec pour nom “isAdult” et qui aura comme valeur “true” si vous avez plus de 18 ans ou “false” dans le cas contraire (boolean).
