Exercice 10

Réaliser un petit gestionnaire de compte tout simple. Voici les différentes consignes:

  • Page d'accueil avec formulaire d'envoi comprenenant le numéro de compte et le montant à envoyer.
  • Un footer avec le copyright "EasyBank" (ou le nom que vous voulez)
  • Un menu avec les liens vers la page "Transferts", "Danger".
  • La page "Comptes" permettra de voir la liste des comptes avec l'argent actuel de celui-ci.
  • La page "Danger" aura un champ dans lequel nous devrons renter un mot de passe (123456789). Si c'est le bon mot de passe est entré cela aura pour effet d'effacer toutes les données de la page.
  • On pourra préciser en plus sur le formulaire si c'est on dépose ou on retire de l'argent.
